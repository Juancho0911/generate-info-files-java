# Generación de Datos - Proyecto Java

## Descripción
Sistema automatizado desarrollado en Java para la gestión, procesamiento y análisis de información de ventas. El programa cumple dos funciones principales: la creación de datos de prueba pseudoaleatorios y la generación de reportes financieros consolidados a partir de múltiples archivos planos.
## Funcionalidad
Genera automáticamente:
- Archivo de productos
- Archivo de vendedores
- Archivos de ventas por vendedor
- Generación Automática
- Creación de catálogo de productos con precios
- Registro de vendedores con datos coherentes
- Generación de archivos de ventas individuales por vendedor.

## Ejecución
Ejecutar la clase:
GenerateInfoFiles -
MAIN

## Requisitos
- Java 8
- Eclipse IDE
