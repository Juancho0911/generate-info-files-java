# Generación de Datos - Proyecto Java

## Descripción
Programa que genera archivos de prueba para un sistema de ventas.

## Funcionalidad
Genera automáticamente:
- Archivo de productos
- Archivo de vendedores
- Archivos de ventas por vendedor

## Ejecución
Ejecutar la clase:
GenerateInfoFiles

## Requisitos
- Java 8
- Eclipse IDE
